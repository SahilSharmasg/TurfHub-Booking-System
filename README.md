# 🏏 TurfHub Booking System

A production-ready full-stack Turf Booking System built using **ASP.NET Core Web API**, **Angular**, **SQL Server**, and **Entity Framework Core**.

This project is being developed as a real-world application to provide online turf booking services with a modern, responsive user interface and a scalable backend architecture.

---

## 📖 Project Overview

TurfHub is a platform where users can search for sports turfs, check available time slots, and book them online. Turf owners can manage their turfs, bookings, pricing, and customer information through a dedicated admin dashboard.

This project is designed following industry best practices and serves as both a production-ready application and a portfolio project.
